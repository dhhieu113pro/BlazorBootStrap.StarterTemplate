# Blazor WebAssembly with Bootstrap Starter Template

This template creates a Blazor WebAssembly application with Bootstrap.

## Usage

Install the template:
```bash
dotnet new install .
```

Create a new project from the template:
```bash
dotnet new blazorwasm-bootstrap -n MyProject
```

## Parameters

- `--Framework`: Target framework (net10.0, net9.0, or net8.0). Default: net10.0

## Example

```bash
dotnet new blazorwasm-bootstrap -n MyBlazorApp --Framework net10.0
```

## Uninstalling

To uninstall the template:
```bash
dotnet new uninstall BlazorBootStrap.StarterTemplate
```
